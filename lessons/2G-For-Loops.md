For Loop 


```js
    for(let i = 0 ; i < 10; i++){
        // repeated code goes here

```


Looping through an array 


```js
    let fruits = ['apple', 'mango' , 'orange' , 'papaya']

    for(let i=0; i<fruits.length; i++;){
        console.log(fruits[i])
    }

```
